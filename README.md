
Luxury Housing Analysis – Bangalore
<h1>Project Overview</h1>
This project analyzes the Luxury Housing Market in Bangalore to uncover insights regarding pricing models, amenity impact, builder performance, and customer conversion trends.
The workflow integrates Python for backend data engineering and Power BI for high-level executive storytelling.
Python (Pandas, NumPy): Data cleaning, normalization, and statistical preprocessing.
Power BI: Interactive dashboarding and geospatial trend analysis.

<H2> Objectives</H2>
Conversion Drivers: Understand specific factors influencing property conversion rates.
Benchmarking: Compare builder performance across the luxury segment.
Market Trends: Identify pricing fluctuations and demand shifts in Bangalore micro-markets.
<h2>Dataset</h2>

The analysis is based on a comprehensive dataset containing:
Identity: Property ID, Builder Name, Location.
Financials: Ticket Price, Revenue.
Attributes: Amenity Score, Configuration (BHK).
Sales Data: Booking Flag (Converted / Not Converted), Sales Channel.

<h2>Data Processing (Python)</h2>
The preprocessing phase involved a rigorous pipeline to ensure data integrity:
Data Cleaning: Handling null values and removing outliers in ticket pricing.
Type Correction: Standardizing categorical data for Power BI compatibility.
Feature Engineering: Creating performance metrics.
